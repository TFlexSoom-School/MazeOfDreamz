## MazeOfDreamz by DreamzCatcherz

### Installation

1. Navigate to the repository where you want to install the project
2. Clone the repo

```
git clone https://github.com/TFlexSoom/MazeOfDreamz.git
```  

### Run Game

Open *MazeOfDreamz/index.html* in your web browser

### Game Controls

First Player
Move: [w][a][s][d]  
Attack: [e]

Second Player
Move: [i][j][k][l]  
Attack: [o]
